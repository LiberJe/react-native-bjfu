# 运行前

## 必要
确保node的环境被正确安装
确保npm被正确安装
确保react-native-cli被正确安装
确保Python被正确安装 //不支持Python3
确保jdk被正确安装，环境变量配置正确
确保Android SDK被正确安装，环境变量正确
确保安装了安卓模拟器，推荐Genymotion

## 可选
yarn Facebook的加速包
cnpm 国内的npm镜像（npm国内可能会出现网络问题）

首先安装依赖
npm install

打开模拟器并确保连接
adb devices //检查设备连接

# 运行
react-native run-android
